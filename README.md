## Facial Emotion Detector App:-

Welcome to the **Facial Emotion Detector** — a Deep Learning powered web app that detects human emotions from facial expressions! This project is built using **TensorFlow**, **Keras**, and deployed using **Streamlit**. It's lightweight, fast, and made with love.

## Features:-

- Real-time emotion detection using an uploaded image
- Clean and interactive UI built with Streamlit
- Custom styling using CSS
- Simple and elegant deployment on Streamlit Cloud

## Model Info :-

- Built using Convolutional Neural Networks (CNNs)
- Trained on a facial emotion dataset
- Model file: `emotion.h5`

## 🗂️ File Structure :-

├── app.py # Main Streamlit app
├── emotion.h5 # Trained DL model
